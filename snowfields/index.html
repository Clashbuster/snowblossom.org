Snowfields

These snow fields are required to do mining with the Snowblossom protocol. They are not required to run a node or use a wallet. To mine with them, you can download them as torrents or generate them locally using the SnowFall program.

All miners (PoolMiner, Arktika, SurfMiner) can read the regular, non-chunked fields.

The chunked versions are only useful if you are using the advanced Arktika miner AND need to separate the field only multiple block devices.
